# Dice Roll Simulator

A simple JavaScript application that simulates rolling dice. Useful for games, learning, or just for fun!

## Features

- Roll one or multiple dice
- Displays random results each time
- Easy-to-use interface

## Getting Started

1. **Clone the repository:**
  ```bash
  git clone https://github.com/yourusername/dice-roll-simulator.git
  ```
2. **Open `index.html` in your browser.**

## Usage

- Click the "Roll" button to simulate rolling the dice.
- View the result instantly on the page.
